## 修改海域省份为内海

- 打开`map/terrain.txt`文件
- 搜索`inland_ocean`
- `terrain_override = { #在这里添加海域省份 }`

如以下所示

```

		color = { 0 0 200 }

		sound_type = sea
		is_water = yes
		inland_sea = yes

		movement_cost = 1.0

		terrain_override = {
```
